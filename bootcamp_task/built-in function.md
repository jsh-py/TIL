## 과제 내장함수 
`is_digit` : 문자열값이 정수값으로 바꿀 수 있으면 TRUE로 반환

`strip()` : 문자열에서 앞뒤공백제거 

`split()` : 문자열을 ()기준에서 리스트로 쪼개줌 

`''.join()` : 리스트를 ''기준에서 문자열로 바꿔줌
```python
a = "     안 녕 하 세 요        "
a.is_digit() >> False
a.strip(' ') >> "안 녕 하 세 요"
a.split(" ") >>  [''....'안','','녕','',...]
''.join(a) >> "안녕하세요"
' '.join(split()) >> "안 녕 하 세 요"

a = "abcdefg"
a = list(a) >> ["a","b","c","d","e","f","g"]
